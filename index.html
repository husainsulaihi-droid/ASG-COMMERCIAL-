API Integrations

Connect live lead sources — Meta Ads, Google & more

f
Meta Lead Ads

Pulls leads from your Facebook & Instagram lead generation campaigns automatically.

✓ Connected
How to get your Access Token
  1. Go to Meta for Developers → Tools → Graph API Explorer
  2. Select your App → click Generate Access Token
  3. Grant permissions: leads_retrieval, pages_manage_ads, pages_read_engagement
  4. Copy the token and paste below
  5. For your Form ID: Meta Ads Manager → Your Campaign → Lead Form → copy the numeric ID from the URL
G
Google Lead Form Extensions

Import leads from Google Ads Lead Form Extensions via Google Sheets sync.

Recommended Setup (Google Sheets Sync)
  1. In Google Ads, go to your campaign → Lead Forms → Download lead data as Google Sheet
  2. Share the sheet publicly (Anyone with link → Viewer)
  3. Copy the Sheet ID from the URL (the long string between /d/ and /edit)
  4. Paste below — the app will read new leads every time you sync
HS
HubSpot Free CRM

Two-way sync — push your ASG leads into HubSpot contacts & deals, or pull HubSpot contacts in as leads.

✓ Connected
How to get your Private App Token
  1. Log in to app.hubspot.com → click your account name (top right) → Account Settings
  2. Go to Integrations → Private Apps → click Create a private app
  3. Name it "ASG Dashboard", then go to the Scopes tab
  4. Enable: crm.objects.contacts.read, crm.objects.contacts.write, crm.objects.deals.read, crm.objects.deals.write, crm.associations.read, crm.associations.write
  5. Click Create app → copy the access token shown
Stage Mapping (ASG → HubSpot)
🆕 New→Appointment Scheduled 📞 Contacted→Qualified to Buy 🤝 Meeting→Presentation Scheduled ✅ Qualified→Decision Maker Bought-In 📄 Proposal→Contract Sent 🏆 Won→Closed Won ❌ Lost→Closed Lost

Real-Time Webhooks

Receive leads instantly the second someone fills your form — requires a live server endpoint.

🔴
No webhook endpoint configured

You need a server URL to receive webhooks. Once your site is hosted (Netlify, Railway, etc.) this endpoint is auto-generated.

How Real-Time Works
  1. You host this app (e.g. Netlify) — a serverless function endpoint is created automatically
  2. Your endpoint URL looks like: https://yoursite.netlify.app/.netlify/functions/meta-webhook
  3. In Meta Ads Manager → Lead Forms → Webhooks → Subscribe to Lead Ads
  4. Paste your endpoint URL + verify token → Meta will send leads instantly 24/7
  5. Every new lead appears in your pipeline within seconds — no manual sync needed
